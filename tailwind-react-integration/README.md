# Tailwind React Integration

This project demonstrates how to integrate **Tailwind CSS** into a **React** application using **Vite**.

## Features

- React app powered by Vite
- Tailwind CSS for styling
- Example of utility-first CSS in React components
- Fast development with hot-reloading

## Project Setup

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/tailwind-react-integration.git
cd tailwind-react-integration
```
